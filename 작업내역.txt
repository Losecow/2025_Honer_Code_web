전체적인 웹사이트 코딩을 맡아서 진행하였다.

모든 페이지(Main, 명예서약서, 섬김이 지도, 명예 게시판, 아너 레벨 테스트)의 전체적인 구성 및 디자인을 맡아서 진행하였다. 각 페이지의 요소 및 컨텐츠를 팀원들과 함께 구상하고, 설계된 내용을 바탕으로 실제 페이지를 제작하는 역할을 맡았다. 설계 과정에서 원활한 소통을 위해 각자 담당할 페이지들을 나눠 반복적인 개발 요청을 확인하고 그에 맞게 수정하는 역할을 수행하였다.

구체적으로 구현한 주요 기능들은 다음과 같다:

**1. 반응형 웹사이트 디자인 및 레이아웃 구현**
- 모든 페이지에 통일된 네비게이션 바와 헤더 적용
- CSS Flexbox와 Grid를 활용한 반응형 레이아웃 구성
- 그라데이션 배경과 모던한 UI/UX 디자인 적용
- 모바일 및 데스크톱 환경 대응

**2. 다국어 지원 시스템 구축**
- 한국어/영어 전환 기능 구현 (KOR/ENG 버튼)
- 모든 페이지의 텍스트, 버튼, 레이블, 플레이스홀더 등 요소별 번역 데이터 관리
- localStorage를 활용한 언어 설정 저장 및 페이지 이동 시에도 언어 상태 유지
- 동적으로 생성되는 콘텐츠(게시판, 퀴즈, 모달 등)에도 번역 적용

**3. 명예서약서 페이지 (명예 서약서)**
- Canvas API를 활용한 실시간 서명 기능 구현
- 이름, 학번, 전공 등 동적 입력 필드 구현
- html2canvas와 jsPDF 라이브러리를 사용한 PDF 다운로드 기능
- A4 용지 크기에 맞춘 PDF 레이아웃 최적화
- 서명 완료 여부에 따른 버튼 활성화/비활성화 처리

**4. 섬김이 지도 페이지 (섬김이 지도)**
- 퍼센트 좌표 기반 인터랙티브 지도 구현
- 지도 상 6개 장소 버튼 배치 및 위치 조정 시스템
- 마우스 호버 시 장소 정보 모달 표시 기능
- 각 장소별 이미지 슬라이더 구현 (단일/다중 이미지 지원)
- 장소별 섬김 활동 설명 및 설명 텍스트 표시

**5. 명예 게시판 페이지 (명예 게시판)**
- 게시글 작성, 조회, 삭제 기능 구현 (CRUD)
- 제목, 작성자, 내용 기반 검색 기능
- 페이지네이션 시스템 구현
- 좋아요 기능 및 댓글 시스템
- 익명 작성 옵션 제공
- localStorage를 활용한 데이터 저장 및 관리

**6. 아너 레벨 테스트 페이지 (아너 레벨 테스트)**
- 상황별 선택지 기반 퀴즈 시스템 구현
- 진행 상황 표시 및 답변 선택 인터페이스
- 점수 계산 알고리즘을 통한 레벨 분류
- 개인 맞춤형 결과 표시 및 실천 조언 제공
- 퀴즈 재시작 기능

**7. 메인 페이지 (Main)**
- 2008년부터 2025년까지 서약식 사진 이미지 슬라이더 구현
- 자동 재생 및 수동 네비게이션(화살표, 도트) 기능
- 이미지 출처 표시 및 유튜브 링크 연결
- 정직 지수 온도계 시각화 컴포넌트
- 성경 구절 표시 섹션
- 아너코드 소개 콘텐츠 레이아웃 구성

**8. 기타 기술적 구현사항**
- Vanilla JavaScript를 사용한 순수 프론트엔드 개발
- 이벤트 리스너를 통한 사용자 인터랙션 처리
- DOM 조작을 통한 동적 콘텐츠 생성 및 업데이트
- 이미지 파일명 인코딩 처리로 Netlify 배포 환경 호환성 확보
- 캐시 버스팅을 위한 스크립트 버전 관리 시스템
- 모달 창 및 애니메이션 효과 구현

각 페이지 간의 데이터 연동 및 사용자 경험 최적화에 중점을 두었으며, 팀원들의 피드백을 반영하여 지속적으로 개선해 나갔다.
